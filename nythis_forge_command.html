NYTHIS Forge Command — README

What Is This

A standalone, Claude-powered multi-agent system for coding, architecture, auditing, project management, and deployment. Five specialized AI agents, one command center. Paste your API key and run.

Setup

  • Get an Anthropic API key at console.anthropic.com
  • Paste your key in the top-right field (starts with sk-ant-)
  • Select your agents and routing mode
  • Type your task and hit Run Agents

Deploy It Yourself

  • Render / Vercel Static: Drop the HTML file into a public folder, deploy as static site. No server needed.
  • GitHub Pages: Push to a repo, enable Pages, done.
  • Local: Open the file directly in any browser — zero install.
  • Self-hosted with key protection: Wrap in a Next.js API route that proxies Claude. Keep your key server-side.

The 5 Agents

  • Architect — System design, tech stack, data model, API contracts, folder structure
  • Coder — Production-ready code, real wiring, real error handling, no shells
  • Auditor — No-fake audit, security review, dead button detection, structured report
  • Project Manager — Pass reports, milestones, readiness estimates, task breakdowns
  • Deployer — Render-first deploy, env vars, migrations, health check, smoke test checklist

Routing Modes

  • Auto Route — Reads your prompt and picks relevant agents automatically
  • Chain Mode — Runs all selected agents together in sequence
  • Single Agent — One agent, full focus

What To Prompt

  • Design a webhook-first n8n router with model lanes and proposal output
  • Write a Next.js API route with Prisma, full validation, typed response
  • Audit my project for fake buttons, dead logic, hardcoded metrics
  • Create a project roadmap with pass reports and readiness estimates
  • Write Render deployment instructions without GitHub integration
  • Build a local-first AI workflow with Ollama and n8n
  • Review this architecture for drift, fake logic, or feature creep
  • Generate a no-fake product audit report for production readiness

NYTHIS Standards Baked In

Every agent call carries the NYTHIS mission: webhook-first, no fake logic, no feature drift, no workaround hell, no fake success. Deployable results only.

Notes

  • Your API key stays in your browser — nothing is stored server-side
  • Output streams token by token in real time
  • Add project context in the Context tab to sharpen every response
  • History keeps last 10 runs — click any entry to restore it
